# Anika X Hannibal

A dual-persona AI system capable of acting as an Expert Interviewer (Anika) or a Professional Therapist (Dr. Hannibal Lecter), providing real-time voice and text interactions.

## Agents

- **Anika (The Interviewer):** A professional, insightful agent designed for career guidance and interview preparation.
- **Dr. Hannibal Lecter (The Therapist):** A calm, refined, and insightful agent focused on providing thoughtful therapeutic sessions.

## Key Features

- **Dual Persona Configuration:** Seamlessly switch between the Interviewer and Therapist modes.
- **Real-Time Voice Interaction:** Provides a natural, conversational experience with distinct voices for each persona.
- **Noise Cancellation:** Ensures clean audio input for accurate voice recognition and interaction.

## Technologies Used

- **Python:** Core programming language.
- **LiveKit Agents Framework:** For handling real-time WebRTC audio sessions and agent state.
- **Google Gemini Realtime API:** The underlying LLM powering the intelligent, low-latency voice responses.
- **DuckDuckGo Search & LangChain:** For real-time web browsing capabilities.
- **Docker:** For containerized, isolated deployment.
